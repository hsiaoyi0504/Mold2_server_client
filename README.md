## Mold2_server_client
### A simple server and client based on [FDA Mold2 program](http://www.fda.gov/ScienceResearch/BioinformaticsTools/Mold2/default.htm)

### Usage
* On server: execute `python3 server/main.py`
* On client: execute `python3 client/main.py`

### Requirement
* Python3
* On server
	* [Bottle](http://bottlepy.org/docs/dev/index.html)
* On client
	* [Requests](http://docs.python-requests.org/en/master/)
